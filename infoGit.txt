git:

git status <= lista de archivos
git add {file name} <= graba localmente in state. onready for commit
git add . <= graba todos los archivos, listos para commit
git commit -m "{description}"  <= send file/s server remote
git branch {branch name} <= create branch
git checkout {branch name} <= go to branch
git branch <= muestra las ramas y me indica donde estoy parado

para mergear:
1) estar en la rama donde quiero aplicar mis cambios. Ej me paro en master y mergeo ahi los cambios de la rama
git branch
git checkout master {rama donde aplicar cambios}
git merge {rama a unir con master}


git branch -d {nombre de rama} <= elimina rama
